# Edite com Agentes — Site

Site publicado em:

https://edite-com-agentes-site.vercel.app/#oferta

## Como editar

O arquivo principal do site é:

```text
index.html
```

Para alterar texto, preço, botão, seção ou link:

1. Abra o arquivo `index.html` no GitHub.
2. Clique no ícone de lápis para editar.
3. Faça a alteração.
4. Clique em **Commit changes**.
5. A Vercel publica automaticamente a nova versão a partir da branch `main`.

## Imagens e logos

As imagens ficam na pasta:

```text
assets/
```

Se trocar uma imagem, mantenha o mesmo nome do arquivo quando possível. Assim o HTML não precisa ser alterado.

## Publicação

Este repositório está conectado ao projeto da Vercel:

```text
edite-com-agentes-site
```

Branch de produção:

```text
main
```

Toda alteração enviada para `main` dispara nova publicação na Vercel.
